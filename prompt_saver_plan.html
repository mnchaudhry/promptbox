User stories

Core — Saving

US1Save any selected text as a prompt with 1 click from the browser extension · extension
US2Add a title, tags, and optional note before saving · extension + web
US3Copy prompt to clipboard instantly without opening the web app · extension
US4Paste a prompt manually (not just selected text) via the extension popup · extension

Core — Browsing & Search

US5Search prompts by keyword across title, body, and tags · web
US6Filter prompts by tag or category · web + extension
US7Browse recent prompts in the extension popup without going to the web app · extension
US8Sort by newest, most used, or alphabetical · web

Core — Management

US9Edit a saved prompt's title, body, and tags · web
US10Delete a prompt with undo (trash + 5s undo) · web
US11Organize prompts into folders or collections · web
US12Export all prompts as JSON or Markdown · web

Auth & Sync

US13Sign up / log in with Google (no passwords) · web + extension
US14Prompts sync in real-time between extension and web app · both
US15Work offline in extension; sync when back online · extension
